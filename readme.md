# 🌐 Site Dev - Portfólio

Este é um projeto de site pessoal/portfólio desenvolvido em **HTML + CSS**, com deploy automático via **Netlify**.

---

## 🚀 Tecnologias
- HTML5
- CSS3
- Git & GitHub
- Netlify (deploy automático)

---

## 🎨 Estrutura
- **Hero (banner topo)**: título, descrição e botão com gradiente.
- **Seções**: alternam fundo claro e cinza, com títulos decorados.
- **Cards**: blocos com sombra e efeito hover.
- **Habilidades**: exibidas em badges arredondados com gradiente e animação hover.
- **Contato**: links estilizados.
- **Rodapé**: animado com fade-in.
